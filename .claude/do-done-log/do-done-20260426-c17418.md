## My refined mental model

Each window in single-screen mode has a **virtual frame** (where it would be if the screen were infinite) and a **visible frame** (virtual ∩ screen). Moves operate on virtual; visible is recomputed. The "absorbed" amount per edge is just `virtualFrame − visibleFrame`. So the only persistent state per window is the virtual frame — absorption is derived, not stored.
